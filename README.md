## Features

- **Task Management** - Organize tasks with Today, Upcoming, Completed, and Inbox views
- **Pomodoro Timer** - Stay focused with built-in Pomodoro technique support
- **Calendar Integration** - Sync with Google Calendar to manage all your events
- **AI-Powered Suggestions** - Get intelligent recommendations for task prioritization
- **Smart Scheduling** - Automatic task prioritization and energy-based scheduling
- **Dark Mode** - Full dark mode support for comfortable viewing

## Tech Stack

- React 19 + TypeScript
- Vite
- Tailwind CSS
- ShadCN UI
- React Query
- React Router

## Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Run linting
npm run lint
```
